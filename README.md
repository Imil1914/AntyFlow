# AntyFlow

Локальный AI-холст: бесконечная доска (как Figma / Miro), где ноды — это заметки,
документы, чаты с ИИ, генерация изображений, презентации, Jupyter-ноутбуки, PDF с
Q&A и встроенные AI-агенты. Стек: **Electron + React + tldraw**.

Всё работает локально на вашей машине, модели подключаются по вашим API-ключам
(или локально через LM Studio / Ollama).

---

## Возможности

- **Заметки и документы** с рендером Markdown.
- **ИИ-чат-ноды** с любой OpenAI-совместимой моделью (свои ключи).
- **Генерация изображений** (через внешний ComfyUI — опционально).
- **Презентации** (слайды 1280×720, экспорт) и **Mermaid-схемы**.
- **Jupyter/Colab-нода** — постоянный Python-kernel прямо в ноде.
- **PDF-нода** — аннотации, локальный RAG и Q&A по документу.
- **Встроенные AI-агенты** (см. ниже): OpenCode, AnythingLLM, OpenScience.
- **Контекст по стрелке** — соедините ноды вектором, и чат увидит их содержимое.
- **Полноэкранный режим** (⛶) и быстрый зум к ноде (**Shift+F**).

---

## Требования

- **Node.js 18+** (обязательно — для сборки и работы агентов).
- **Git** (нужен агенту AnythingLLM — он клонируется автоматически).
- **Windows** (сборка настроена под Windows/NSIS; можно адаптировать под другие ОС).

---

## Установка и запуск

```bash
git clone https://github.com/Imil1914/AntyFlow.git
cd AntyFlow
npm install
```

**Режим разработки** (быстрый старт, с hot-reload):
```bash
npm run dev
```

**Собрать установщик** (`.exe` под Windows, в папке `release/`):
```bash
npm run dist
```

---

## Первый запуск: вводим свои API-ключи

1. Откройте **настройки провайдеров** (командное меню **Ctrl+K** → «Настройки провайдеров»).
2. Включите нужного провайдера, впишите **baseURL** и **свой API-ключ**. Поддерживается
   любой OpenAI-совместимый эндпоинт:
   - OpenRouter — `https://openrouter.ai/api/v1`
   - LM Studio (локально, без ключа) — `http://127.0.0.1:1234/v1`
   - или любой другой OpenAI-совместимый сервис.
3. Для нод без списка моделей укажите модели вручную через запятую.

Ключи хранятся **локально** (в профиле пользователя), в репозиторий не попадают.

## Встроенные AI-агенты

Агенты **не входят в этот репозиторий** — Flow ставит/подтягивает их при первом
использовании. Нужно только их разрешить (интернет + Node.js).

| Агент | Что делает | Как ставится |
|---|---|---|
| **AnythingLLM** | RAG-ассистент с базой документов | Flow сам клонирует и собирает при первом запуске ноды (~несколько минут, ~1 ГБ). Свой ключ LLM задаётся в его интерфейсе. |
| **OpenScience** | Научный AI-воркбенч (`@synsci/openscience`) | `npm i -g @synsci/openscience`. Ключ/модель — в его веб-интерфейсе или конфиге. |
| **OpenCode** | Агент для кода в реальном терминале | Установите `opencode` (см. его репозиторий). Запускается в PTY внутри ноды. |

> **ComfyUI не входит** в проект и не требуется. Генерация изображений — опциональная
> фича: работает, только если вы отдельно поднимете свой ComfyUI и укажете его адрес
> в настройках. Без него всё остальное работает как обычно.

---

## Где что хранится (не коммитится)

- Ключи провайдеров, доски, настройки — в профиле пользователя (`%APPDATA%/flow` на Windows).
- Агенты и их данные — там же (`anythingllm/`, партиции webview) и в глобальном npm / `~/.config`.

Поэтому обновление/переустановка приложения настройки **не стирает**.

---

## Лицензии

Flow — ваш код. Встроенные агенты (AnythingLLM, OpenScience, OpenCode, ComfyUI) —
отдельные проекты со своими лицензиями; здесь только код интеграции с ними.
