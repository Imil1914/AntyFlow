# Flow Sync Server — real-time совместные доски

Сервер, который позволяет нескольким людям работать с **одной доской Flow в реальном
времени** (как в Figma): видно курсоры и правки друг друга. Построен на официальном
[`@tldraw/sync`](https://tldraw.dev/docs/sync) и разворачивается на **Cloudflare Workers**.

## Что нужно один раз настроить

1. **Аккаунт Cloudflare** и **план Workers Paid** (~$5/мес).
   Durable Objects (на них держится комната) доступны только на платном плане.

2. Установить зависимости и залогиниться:
   ```bash
   cd sync-server
   npm install
   npx wrangler login
   ```

3. Создать R2-бакет для снапшотов досок (переживают перезапуск комнаты):
   ```bash
   npx wrangler r2 bucket create flow-sync-rooms
   ```
   > Если не хочешь R2 — можно убрать блок `[[r2_buckets]]` из `wrangler.toml`
   > и переписать сохранение на `state.storage`, но у него лимит 128 КБ на запись.

4. Задеплоить:
   ```bash
   npx wrangler deploy
   ```
   Wrangler напечатает адрес воркера, например:
   `https://flow-sync.ТВОЙ-АккаунТ.workers.dev`

## Как подключить приложение

В Flow → меню доски (стрелка у названия) → **⚙ Сервер синхронизации** → впиши адрес
воркера в формате WebSocket, заменив `https` на `wss`:

```
wss://flow-sync.ТВОЙ-АккаунТ.workers.dev
```

Дальше:
- **🌐 Сделать общей** — текущая доска становится общей, приложение показывает **ID комнаты**.
- Передай этот ID (и адрес сервера) второму человеку.
- У него: **🔗 Подключиться** → тот же адрес сервера + ID комнаты.

Теперь оба редактируют одну доску в реальном времени.

## Что синхронизируется, а что нет

**Синхронизируется** (живёт в документе доски): расположение и содержимое всех нод —
канбаны, бэклоги, заметки, текст, история ИИ-чатов, ответы ИИ в карточках, картинки.

**НЕ синхронизируется** (привязано к машине каждого): живые терминалы OpenCode,
Python-kernel, webview (AnythingLLM/OpenScience), а также запуск локальных моделей —
когда кто-то жмёт «🤖 Сделать», запрос идёт на его LM Studio/ключи; общий только текст-результат.

## Локальная разработка сервера

```bash
npx wrangler dev
```
Поднимет сервер на `http://localhost:8787`; в Flow укажи `ws://localhost:8787`.

## Стоимость и безопасность

- Cloudflare Workers Paid — ~$5/мес; R2 — есть бесплатный лимит.
- Доступ к доске = знание адреса сервера + ID комнаты. Для приватности используй
  длинные случайные ID (приложение генерирует такие) и не публикуй их.
- Хочешь ограничить круг — не публикуй адрес воркера; при желании добавь проверку
  токена в `worker.ts` (в `uri` можно прокидывать短-живущий токен, см. доки tldraw sync).
